Wichtiger Hinweis:

Sie haben erfolgreich Ihr pidoco Projekt von unserem Server heruntergeladen.
Um damit offline arbeiten zu können müssen Sie das Archiv zunächst entpacken.

Windows Nutzer:
Rechtsklick auf der Zip Datei, um diese zu extrahieren und wählen Sie dann einen Ort für die Dateien.
Danach gehen sie zu diesem Ordner und öffnen die Dateien von dort.

Ein weiterer Hinweis:
Öffnen Sie die index.html Datei in Ihrem Browser und wählen Sie eine Seite zur Ansicht im skizzierten oder normalen Modus.

Falls Sie die Dateien auf Ihren eigenen Web Server hochladen möchten, achten Sie bitte darauf, 
dass die xhtml Dateien mit dem mime type 'application/xhtml+xml' ausgeliefert werden.
